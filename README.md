# 小丑牌

Balatro 风格的网页扑克游戏。玩家通过出牌组成牌型累积分数，购买小丑牌增强能力，依次挑战三个盲注关卡。

## 功能

- 9 种牌型识别与计分（高牌 → 同花顺）
- 6 张小丑牌（Joker），各有独特加成效果
- 3 关递增盲注（300 / 500 / 800 分）
- 出牌动效：飞牌、逐张高亮、浮字、公式爆出、补牌发牌
- AI 出牌助手（本地枚举最优组合）
- 设置面板：动画速度、公式预览，持久化到 localStorage

## 安装与运行

需要 Node.js 18+。

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

启动后打开 http://localhost:5173

## 玩法

1. 从 8 张手牌中选 1-5 张，点「出牌」组成牌型得分
2. 每关 4 手内累积分数达到目标则通关，进入商店
3. 商店可购买小丑牌提升后续每手的倍率/筹码
4. 三关全部通过即胜利
